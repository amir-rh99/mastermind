# MasterMind

A classic code-breaking puzzle game built with React, TypeScript, and Tailwind CSS.

Crack the secret color code using logic and deduction.

**[Play Online →](https://amir-rh99.github.io/mastermind/)**

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v4
- Lucide icons

## Development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
